

# RPA 自动化流程平台

这是一个基于 Vue.js 和 Element Plus 构建的企业级 RPA（机器人流程自动化）管理平台。该平台提供了完整的流程设计、任务管理、数据处理和系统管理功能。

## 功能特性

### 核心模块
- **仪表盘 (Dashboard)** - 实时数据统计和可视化展示
- **客户管理 (Client Management)** - 客户信息维护和管理
- **任务管理 (Task Management)** - RPA 任务创建、编辑和执行管理
- **流程设计 (Flow Design)** - 可视化流程配置和 AI 优化功能
- **流程列表 (Flow List)** - 流程模板管理和维护
- **数据采集 (Data Collect)** - 数据抓取和收集功能
- **数据解析 (Data Parse)** - 数据格式解析和处理
- **数据处理 (Data Process)** - 数据清洗和预处理
- **数据查询 (Data Query)** - 数据查询和过滤
- **执行记录 (Execution Record)** - 任务执行日志和结果查看

### 系统管理
- **用户管理** - 系统用户账号管理
- **角色管理** - 权限角色配置
- **资源管理** - 系统资源分配
- **个人中心** - 用户资料和设置

## 技术栈

- **前端框架**: Vue.js 3
- **UI 组件库**: Element Plus
- **图表库**: ECharts
- **构建工具**: Vite
- **HTTP 请求**: Axios

## 安装配置

### 环境要求
- Node.js 16+
- npm 8+ 或 yarn 1.22+

### 安装步骤

1. 安装依赖：
```bash
npm install
```

2. 配置环境变量：
```bash
cp .env.example .env
```

3. 修改 `.env` 文件中的 API 配置：
```
VITE_API_BASE_URL=你的API地址
```

4. 启动开发服务器：
```bash
npm run dev
```

5. 构建生产版本：
```bash
npm run build
```

## 主要页面说明

### 登录页面
支持用户名密码登录，提供记住密码功能。

### 仪表盘
展示关键业务指标，包括：
- 任务总数统计
- 执行成功率
- 数据处理量
- 近期任务列表

### 任务管理
- 创建新任务（选择流程、配置参数）
- 任务列表查询和筛选
- 任务详情查看
- 任务执行控制

### 流程设计
- 目标 URL 配置
- 字段配置（名称、描述、必填项）
- AI 智能优化功能
- 实时预览

### 执行记录
- 按时间查询执行历史
- 查看执行结果和日志
- 错误信息详情

## 注意事项

1. 确保后端 API 服务正常运行
2. 登录后会自动获取用户权限信息
3. 某些功能需要相应权限才能访问
